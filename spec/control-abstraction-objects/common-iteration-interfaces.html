An interface is a set of object property keys whose associated values match a specific specification. Any object that provides all the properties of an interface in conformance to the interface’s specification conforms to that interface. An interface isn’t represented by a single object and there may be many distinctly implemented objects that conform to any interface. An individual object may conform to multiple interfaces.
