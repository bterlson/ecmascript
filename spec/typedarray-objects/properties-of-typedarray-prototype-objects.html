The value of the [[Prototype]] internal slot of a TypedArray prototype object is the standard built-in %TypedArrayPrototype% object (22.2.3). A TypedArray prototype object is an ordinary object. It does not have a [[ViewedArrayBuffer]] or or any other of the internal slots that are specific to TypedArray instance objects.

The value of TypedArray.prototype.BYTES_PER_ELEMENT is the Number value of the Element Size value specified in Table 36 for TypedArray.

This property has the attributes { [[Writable]]: false, [[Enumerable]]: false, [[Configurable]]: false }.

The initial value of a TypedArray.prototype.constructor is the corresponding standard built-in TypedArray constructor.
