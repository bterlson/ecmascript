The abstract operation FunctionInitialise requires the arguments: a function object F, kind which is one of (Normal, Method, Arrow), a parameter list production specified by ParameterList, a body production specified by Body, a Lexical Environment specified by Scope, and optionally, a property key methodName and an object homeObject. FunctionInitialise performs the following steps:

  1. Let len be the ExpectedArgumentCount of ParameterList.
  2. Let strict be the vale of Fâ€™s [[Strict]] internal slot.
  3. Let status be the result of DefinePropertyOrThrow(F, "length", PropertyDescriptor{[[Value]]: len, [[Writable]]: false, [[Enumerable]]: false, [[Configurable]]: true}).
  4. ReturnIfAbrupt(status).
  5. If strict is true, then
    1. Let status be the result of the AddRestrictedFunctionProperties abstract operation with argument F.
    2. ReturnIfAbrupt(status).
  6. Set the [[Scope]] internal slot of F to the value of Scope.
  7. Set the [[FormalParameters]] internal slot of F to ParameterList .
  8. Set the [[Code]] internal slot of F to Body.
  9. If the homeObject argument was provided, then
    1. Set the [[HomeObject]] internal slot of F to homeObject.
    2. Set the [[MethodName]] internal slot of F to methodName.
  10. If kind is Arrow, then set the [[ThisMode]] internal slot of F to lexical.
  11. Else if strict is true, then set the [[ThisMode]] internal slot of F to strict.
  12. Else set the [[ThisMode]] internal slot of F to global.
  13. Return F.
