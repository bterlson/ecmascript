The production ClassAtom evaluates by evaluating ClassAtom to obtain a CharSet and returning that CharSet.

The production ClassAtom NonemptyClassRangesNoDash evaluates as follows:

  1. Evaluate ClassAtom to obtain a CharSet A.
  2. Evaluate NonemptyClassRangesNoDash to obtain a CharSet B.
  3. Return the union of CharSets A and B.

The production ClassAtom - ClassAtom ClassRanges evaluates as follows:

  1. Evaluate the first ClassAtom to obtain a CharSet A.
  2. Evaluate the second ClassAtom to obtain a CharSet B.
  3. Evaluate ClassRanges to obtain a CharSet C.
  4. Call CharacterRange(A, B) and let D be the resulting CharSet.
  5. Return the union of CharSets D and C.

The abstract operation CharacterRange takes two CharSet parameters A and B and performs the following:

If A does not contain exactly one character or B does not contain exactly one character then throw a SyntaxError exception.

Let a be the one character in CharSet A.

Let b be the one character in CharSet B.

Let i be the character value of character a.

Let j be the character value of character b.

If i > j then throw a SyntaxError exception.

Return the set containing all characters numbered i through j, inclusive.
