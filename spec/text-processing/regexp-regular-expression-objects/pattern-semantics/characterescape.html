The production ControlEscape evaluates by returning the character according to Table 41 .

The production c ControlLetter evaluates as follows:

  1. Let ch be the character represented by ControlLetter.
  2. Let i be ch's character value.
  3. Let j be the remainder of dividing i by 32.
  4. Return the character whose character value is j.

The production HexEscapeSequence evaluates by evaluating the CV of the HexEscapeSequence (see 11.8.4) and returning its character result.

The production RegExpUnicodeEscapeSequence evaluates by returning the result of evaluating RegExpUnicodeEscapeSequence .

The production IdentityEscape evaluates by returning the character represented by IdentityEscape.

The production LeadSurragate \u TrailSurragate evaluates as follows:

  1. Let lead be the result of evaluating LeadSurragate.
  2. Let trail be the result of evaluating TrailSurragate.
  3. Let cp be the UTF16Decode(lead, trail).
  4. Return the character whose character value is cp.

The production u Hex4Digits evaluates by evaluating the CV of Hex4Digits (see 11.8.4.2) and returning its character result.

The production u{ HexDigits } evaluates by evaluating the MV of the HexDigits (see 11.8.3.1) and returning its character result.

The production Hex4Digits evaluates by evaluating the CV of Hex4Digits (see 11.8.4.2) and returning its character result.
