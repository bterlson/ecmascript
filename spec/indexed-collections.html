Array objects are exotic objects that give special treatment to a certain class of property names. See 9.4.2 for a definition of this special treatment.

An Array object, O, is said to be sparse if the following algorithm returns true:

  1. Let len be the result of Get(O, "length").
  2. For each integer i in the range 0≤i<ToUint32(len)
    1. Let elem be the result of calling the [[GetOwnProperty]] internal method of O with argument ToString(i).
    2. If elem is undefined, return true.
  3. Return false.
